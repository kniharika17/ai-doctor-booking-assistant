# AI Doctor Appointment Booking Assistant

## Overview
An AI-powered chatbot that supports doctor appointment booking using conversational AI, RAG, and LLMs.

## Features
- Chat-based UI using Streamlit
- Conversational booking with slot filling
- PDF-based RAG (Retrieval Augmented Generation)
- SQLite database for bookings
- Email confirmation via Gmail SMTP
- Admin dashboard to view bookings
- Input validation and graceful error handling

## Tech Stack
- Python
- Streamlit
- LangChain
- OpenAI
- SQLite

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
